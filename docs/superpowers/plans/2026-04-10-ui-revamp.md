# MediNetHub UI Revamp Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rebuild the MediNetHub frontend with a coherent design system (Navy + Teal palette, Inter typography, Lucide icons), collapsible sidebar navigation, and unified English language — migrating all 30+ templates from scattered inline CSS to centralized stylesheets.

**Architecture:** Foundation-first approach. Phase 1 creates the design system CSS files and base layout templates. Phase 2 migrates each page one-by-one onto the new system, removing inline styles and translating all text to English. Phase 3 adds new feature UIs.

**Tech Stack:** Django templates, Bootstrap 5 (kept as base), CSS custom properties, Inter + JetBrains Mono (Google Fonts), Lucide Icons (CDN), vanilla JS for sidebar toggle.

**Design Spec:** `docs/superpowers/specs/2026-04-10-ui-revamp-design.md`

### Task 4: Create sidebar.js — Collapse/Expand Logic

**Files:**
- Create: `MediNetHub/static/js/sidebar.js`

- [ ] **Step 1: Create the sidebar JS**

```javascript
/**
 * MediNetHub Sidebar - Collapse/Expand logic
 * Persists state in localStorage.
 */
(function() {
  'use strict';

  var STORAGE_KEY = 'medinet-sidebar-expanded';

  function initSidebar() {
    var sidebar = document.getElementById('sidebar');
    var toggle = document.getElementById('sidebar-toggle');

    if (!sidebar || !toggle) return;

    // Restore state from localStorage
    var savedState = localStorage.getItem(STORAGE_KEY);
    if (savedState === 'true') {
      sidebar.classList.add('expanded');
    }

    // Toggle on click
    toggle.addEventListener('click', function() {
      sidebar.classList.toggle('expanded');
      var isExpanded = sidebar.classList.contains('expanded');
      localStorage.setItem(STORAGE_KEY, isExpanded);
    });
  }

  // Initialize when DOM is ready
  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', initSidebar);
  } else {
    initSidebar();
  }
})();
```
